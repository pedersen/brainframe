# Welcome to BrainFrame

This is the **tutorial engram** — a small, read-only set of notes that ships
inside the app to show you around. You can read it, but you cannot edit or
delete it; it is always here and always pristine.

## What an engram is

An *engram* is BrainFrame's name for a self-contained set of markdown notes
(plus their assets) that the app manages as one unit. A brain holds many
engrams: this tutorial is one, the help reference is another, and the notes
you create live in engrams of your own.

## Where to go next

- Read [Taking notes](notes/first-note.md) for the shape of a note.
- When you are ready, create your own engram and start writing.
