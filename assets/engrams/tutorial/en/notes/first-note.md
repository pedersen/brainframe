# Taking notes

A note is just a markdown file inside an engram. This file lives at
`notes/first-note.md` — a folder named `notes/` with one note in it.

## Markdown basics

- Use `#` headings to structure a note.
- Link to another note with `[label](path.md)`.
- Emphasis is `*italic*` and `**bold**`.

That is the whole model: plain files you own, in folders you arrange.
