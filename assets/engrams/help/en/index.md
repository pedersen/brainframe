# BrainFrame help

This is the **help engram** — reference documentation that ships with the app.
Like the tutorial, it is read-only. Unlike the tutorial, it is meant to be
consulted while you work: open it as a reader overlay without leaving the
engram you are editing.

## Contents

- [Markdown syntax](markdown-syntax.md) — the formatting BrainFrame renders.

## Getting help

BrainFrame is open source. If something here is unclear or wrong, the project
welcomes corrections.
