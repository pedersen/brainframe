# Markdown syntax

BrainFrame notes are plain markdown. This page lists the formatting the reader
understands.

## Headings

Start a line with one to six `#` characters:

```markdown
# Title
## Section
### Subsection
```

## Inline formatting

- `*italic*` renders as *italic*.
- `**bold**` renders as **bold**.
- `` `code` `` renders as inline code.

## Links and lists

- Link with `[label](target.md)`.
- Bullet lists start each line with `-`.
